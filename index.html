Sobre mim

Sou físico e mestre em Astronomia pela USP, com experiência em análise de dados científicos usando Python e SQL. Atualmente trabalho com projetos de Ciência de Dados, focando em:

  • Análise estatística e visualização de dados
  • Limpeza e preparação de datasets
  • Desenvolvimento de pipelines de dados

Como hobby, crio bots automatizados que publicam conteúdo em redes sociais, incluindo imagens astronômicas (NASA APOD), dados meteorológicos e fotos de pets.

Projetos Destacados

Monitoramento COVID-19 em Tempo Real

Sistema automatizado que coleta, processa e visualiza dados atualizados diariamente da Organização Mundial da Saúde sobre a pandemia no Brasil. O pipeline inclui:

  • Coleta automática de dados da OMS
  • Processamento com Python (Pandas, Matplotlib)
  • Visualização interativa de tendências
  • Comparação com dados históricos

Análise Histórica da COVID-19 no Brasil

Estudo abrangente da evolução da pandemia no Brasil, com dados desde 2020 até 2023. O projeto inclui:

  • Análise temporal de casos e óbitos
  • Comparação entre estados brasileiros
  • Cálculo de taxas de letalidade
  • Visualização de padrões e tendências

Análise de dados da sobredensidade TriAnd com dados do espectrógrafo GRACES utilizando Python, resultando em publicação no The Astrophysical Journal. Processamento de grandes volumes de dados astronômicos com análise estatística avançada.
