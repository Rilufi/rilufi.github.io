PetBots - Gatos e Cachorros Automatizados

Bots que postam diariamente imagens fofas de pets nas redes sociais

@boturinsta @doglufi @boturitter @catdogsky.bsky.social

Desenvolvi uma coleção de bots automatizados que publicam diariamente imagens encantadoras de gatos e cachorros em diferentes redes sociais, cada um adaptado às especificações técnicas e melhores práticas de cada plataforma.

Gatobot - Twitter/X e Bluesky

Este bot publica automaticamente imagens de gatos no Twitter/X e Bluesky, com interações inteligentes através de hashtags.

Principais Funcionalidades:

  • Postagem Automática:
    • Publica imagens de gatos diariamente
    • Gera legendas criativas usando IA (Gemini)
    • Formata conteúdo específico para cada plataforma
  • Interação Inteligente:
    • Identifica e segue posts com hashtags relacionadas a gatos
    • Curtir e repostar conteúdo relevante da comunidade
    • Filtros para evitar conteúdo inapropriado
  • Gerenciamento de Limites:
    • Controle de rate limits das APIs
    • Backoff exponencial para requisições
    • Registro de interações para evitar duplicações

Tecnologias Utilizadas:

Python Tweepy Bluesky API Google Gemini API The Cat API

Boturinsta - Gatos no Instagram

Bot dedicado a postar imagens diárias de gatos no Instagram com legendas criativas geradas por IA.

Principais Funcionalidades:

  • Aquisição de Imagens:
    • Utiliza a The Cat API para imagens de alta qualidade
    • Download e processamento das imagens
  • Geração de Conteúdo:
    • Legendas criativas geradas pelo Google Gemini
    • Hashtags relevantes para alcance orgânico
    • Formatação específica para o Instagram
  • Publicação Automatizada:
    • Autenticação segura com gerenciamento de sessão
    • Tratamento de erros e notificações via Telegram
    • Backoff exponencial para tentativas de postagem

Tecnologias Utilizadas:

Python Instagrapi Google Gemini API The Cat API Telegram Bot

Infraestrutura Compartilhada

Todos os bots utilizam:

  • APIs de Imagens: The Cat API e The Dog API para imagens de alta qualidade
  • IA Generativa: Google Gemini para criação de legendas criativas e texto alternativo
  • Monitoramento: Notificações via Telegram para erros e status
  • Gerenciamento de Sessão: Autenticação persistente para evitar novos logins
  • Tratamento de Erros: Backoff exponencial para requisições e tentativas de postagem

Implantação

Os bots são executados diariamente usando:

  • Hospedagem em nuvem para disponibilidade 24/7
  • Variáveis de ambiente para armazenamento seguro de credenciais
  • Agendamento de tarefas para postagens em horários estratégicos
  • Armazenamento de estado para evitar duplicação de conteúdo

Estes bots automatizados trazem alegria diária para amantes de animais, compartilhando imagens fofas de gatos e cachorros com legendas criativas em múltiplas plataformas.
